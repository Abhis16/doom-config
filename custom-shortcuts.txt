Custom shortcuts (Doom Emacs)

Global (leader) shortcuts:
- SPC o a: Open Org agenda dispatcher.
- SPC o c: Open Org capture menu.
- SPC o t: Open todo.org.
- SPC o n: Open notes.org.
- SPC o j: Journal prefix (see Journal section below).
- SPC o e: Essays prefix (see Essays section below).
- SPC o p: Open research/papers.org.
- SPC o 5: Open top 5 per category view (quick access).

Todo capture shortcuts (SPC o c then):
- t p: Capture personal todo (auto-tagged :personal:).
- t r: Capture research todo (auto-tagged :research:).
- t s: Capture school/admin todo (auto-tagged :school:).

Custom agenda views (SPC o a then):
- a: Default agenda (week view with deadlines and scheduled items).
- t: Top 5 per category (5 closest deadlines for each tag).
- p: All personal tasks (sorted by deadline).
- r: All research tasks (sorted by deadline).
- s: All school/admin tasks (sorted by deadline).

Org mode (local leader in org buffers):
- SPC m s: Schedule a task (org-schedule).
- SPC m d: Add a deadline (org-deadline).
- SPC m t: Toggle TODO state (org-todo).
- SPC m q: Set tags (fast-select with p/r/s).
- SPC m A: Archive all DONE tasks to Completed section.

Habit tracking (SPC o h):
- SPC o h h: Open habit review view (main interface).
- SPC o h f: Open habits.org (edit habit definitions).
- SPC o h d: Finalize today's habits (manual, before midnight).
- SPC o h w: Finalize this week's habits (manual, before Sunday midnight).
- SPC o h c: Cancel habit for date range.
- SPC o h u: Uncancel habit range.
- SPC o h v: View all cancellation ranges.
- SPC o h l: View recent habit logs.

In habit review buffer:
- RET or SPC: Cycle habit state (incomplete -> completed -> cancelled).
- c: Mark habit as completed.
- i: Mark habit as incomplete.
- x: Mark habit as cancelled.
- g: Refresh the view.
- q: Quit habit review.

Journal (SPC o j):
- SPC o j j: New journal entry (add timestamped entry for today).
- SPC o j n: New journal entry (alias for j j).
- SPC o j c: Open current week's journal file.
- SPC o j r: Read any week's journal (date picker).
- SPC o j l: List all journal weeks (selection menu).

Essays (SPC o e):
- SPC o e e: New essay (prompts for title, creates file with metadata).
- SPC o e n: New essay (alias for e e).
- SPC o e l: List essays (select from existing essays).
- SPC o e o: Open essays folder in dired.

Website workflow (SPC W - capital W):
- SPC W w: Open website project.
- SPC W n: New blog post (prompts for title, creates file with frontmatter).
- SPC W e: Edit existing post (select from list).
- SPC W p: Preview site (starts zola serve).
- SPC W d: Deploy site (git add, commit, push).
- SPC W b: Build site (zola build).
- SPC W o: Open writings directory.

LaTeX workflow (in .tex buffers):
- SPC m v: View PDF in pdf-tools (integrated Emacs viewer).
- SPC m c: Compile document.
- SPC m a: Run all commands (save, compile, view).
- Save file: Auto-compiles PDF via latexmk.

Insert mode:
- C-[: Exit insert mode to normal mode (evil-normal-state).
